Audio.h:
This file declares the templates and their function

Audio.cpp:
This file defines all the templates methods and special functions of the Audio class

main.cpp:
Is the driver file where all operations are executed. You can type in after you make "./samp.exe --h" to get help on how to structure your command line arguments. And if you enter it in incorrectly it also displays this menu.

AudioUnitTests.cpp:
This file is where unit testing takes place. You can execute all this tests using "./test.exe" to run the unit tests after using "make" or "make test".
